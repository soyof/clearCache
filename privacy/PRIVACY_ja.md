# プライバシーポリシー

**最終更新日：2025 年 10 月 21 日**

---

## はじめに

「キャッシュクリーナーアシスタント」（以下「本拡張機能」）をご利用いただきありがとうございます。私たちはお客様のプライバシーとデータセキュリティを非常に重視しています。本プライバシーポリシーは、本拡張機能がどのように情報を収集、使用、保存、保護するかをご理解いただくためのものです。

**重要なお約束：本拡張機能は、個人を特定できる情報を外部サーバーに収集、保存、送信することはありません。すべてのデータ処理はお客様のローカルデバイス上で行われます。**

---

## 1. 情報収集

### 1.1 収集しない情報

本拡張機能は以下の情報を**一切収集しません**：

- ❌ 個人を特定できる情報（氏名、メールアドレス、電話番号など）
- ❌ 閲覧履歴
- ❌ ウェブサイト訪問データ
- ❌ Cookie の内容
- ❌ フォームデータ
- ❌ パスワードまたはログイン認証情報
- ❌ 位置情報
- ❌ デバイス識別子
- ❌ IP アドレス

### 1.2 ローカルに保存されるデータ

本拡張機能は、パーソナライズされた体験を提供するために、以下の設定情報のみをお客様のローカルブラウザに保存します：

| データタイプ           | 目的                               | 保存場所                   |
| ---------------------- | ---------------------------------- | -------------------------- |
| 言語設定               | 選択したインターフェース言語を記憶 | ローカルブラウザストレージ |
| テーマ設定             | ダーク/ライトテーマの選択を保存    | ローカルブラウザストレージ |
| 通知設定               | 通知のオン/オフ状態を記憶          | ローカルブラウザストレージ |
| クリーニングオプション | クリーニング設定を保存             | ローカルブラウザストレージ |
| 定期クリーニング計画   | 自動クリーニング設定を保存         | ローカルブラウザストレージ |

**これらのデータはお客様のデバイスにのみ保存され、サーバーにアップロードされることはありません。**

## 2. 権限の説明

本拡張機能は、コア機能を提供するために以下のブラウザ権限を必要とします。これらの権限は宣言された目的にのみ使用することをお約束します：

### 2.1 必要な権限

| 権限               | 目的                                                 | データ処理方法                                       |
| ------------------ | ---------------------------------------------------- | ---------------------------------------------------- |
| `storage`          | ユーザー設定と環境設定を保存                         | ローカルストレージのみ、アップロードなし             |
| `browsingData`     | キャッシュ、Cookie などの閲覧データをクリア          | 削除のみ実行、内容は読み取らない                     |
| `tabs`             | クリーニング操作を実行するために現在のタブ情報を取得 | URL ドメインのみ読み取り、完全な URL は記録しない    |
| `activeTab`        | 現在のタブでリロード操作を実行                       | リフレッシュのみトリガー、ページ内容にアクセスしない |
| `downloads`        | ダウンロード履歴をクリア                             | 削除のみ実行、ファイル内容にアクセスしない           |
| `history`          | 閲覧履歴をクリア                                     | 削除のみ実行、履歴内容を読み取らない                 |
| `cookies`          | ウェブサイトの Cookie をクリア                       | 削除のみ実行、Cookie 値を読み取らない                |
| `unlimitedStorage` | ユーザー設定を保存（サイズ制限なし）                 | 設定データのみ保存、ユーザー情報は収集しない         |
| `contextMenus`     | コンテキストメニューにクイックアクションを追加       | メニューオプションのみ提供、データは収集しない       |
| `notifications`    | 操作完了通知を表示                                   | ローカルでのみ表示、外部に送信しない                 |
| `scripting`        | ページでクリーニングスクリプトを実行                 | LocalStorage/SessionStorage のみクリア               |
| `<all_urls>`       | すべてのウェブサイトでクリーニング操作を許可         | クリーニングのみに使用、ページ内容にアクセスしない   |

### 2.2 権限使用の原則

- ✅ **最小権限の原則**：機能実装に必要な権限のみをリクエスト
- ✅ **透明な使用**：すべての権限の目的は本文書で明確に説明
- ✅ **ローカル処理**：すべてのデータ処理はローカルで完了
- ✅ **権限の乱用なし**：宣言された目的以外には権限を使用しない

## 3. データの使用

### 3.1 データ処理方法

本拡張機能のすべての機能はお客様のローカルデバイス上で実行されます：

1. **キャッシュクリーニング**：ブラウザ API を直接呼び出してキャッシュを削除、キャッシュ内容は読み取らない
2. **Cookie クリーニング**：Cookie を直接削除、Cookie 値を読み取ったり記録したりしない
3. **ストレージクリーニング**：LocalStorage と SessionStorage をクリア、ストレージ内容にアクセスしない
4. **履歴クリーニング**：閲覧履歴を削除、履歴データを読み取ったり記録したりしない
5. **ページリロード**：ページのリフレッシュをトリガー、ページ内容にアクセスしない

### 3.2 実行しない操作

本拡張機能は**決して以下のことを行いません**：

- ❌ お客様のデータを外部サーバーに送信する
- ❌ お客様の情報を第三者と共有する
- ❌ お客様の閲覧行動を追跡する
- ❌ お客様の使用習慣を分析する
- ❌ 広告を表示したりマーケティングを行う
- ❌ お客様のデータを販売またはレンタルする

## 4. データセキュリティ

### 4.1 セキュリティ対策

お客様のデータセキュリティを保護するために、以下の対策を講じています：

- 🔒 **ローカルストレージ**：すべてのデータはお客様のデバイスにのみ保存
- 🔒 **ネットワーク送信なし**：拡張機能にはネットワークリクエストコードが含まれていない
- 🔒 **外部依存なし**：第三者サービスや CDN に依存しない
- 🔒 **オープンソースの透明性**：ソースコードは公開されており、レビュー可能
- 🔒 **権限の分離**：ブラウザのセキュリティサンドボックスメカニズムに従う

### 4.2 データの削除

以下の方法で、本拡張機能が保存したすべてのデータをいつでも削除できます：

1. **拡張機能のアンインストール**：アンインストール後、すべてのローカル設定が自動的に削除されます
2. **設定のリセット**：拡張機能の設定で「デフォルト設定に戻す」をクリック
3. **ブラウザデータのクリア**：ブラウザの設定から拡張機能データをクリア

## 5. 第三者サービス

### 5.1 第三者サービスなし

本拡張機能は、以下を含む**いかなる第三者サービスも使用しません**：

- ❌ 分析サービス（Google Analytics など）
- ❌ 広告ネットワーク
- ❌ クラウドストレージサービス
- ❌ ソーシャルメディアプラグイン
- ❌ エラー追跡サービス

### 5.2 外部接続なし

本拡張機能は**いかなる外部ネットワーク接続も確立しません**：

- ❌ 当社のサーバーに接続しない
- ❌ 第三者 API に接続しない
- ❌ 外部リソースを読み込まない
- ❌ テレメトリデータを送信しない

## 14. よくある質問（FAQ）

### Q1: この拡張機能は閲覧履歴を収集しますか？

**A:** いいえ。この拡張機能は、お客様が明示的に要求した場合にのみ閲覧履歴を削除し、履歴内容を読み取ったり収集したりしません。

### Q2: この拡張機能は Cookie を読み取りますか？

**A:** いいえ。この拡張機能は、お客様がクリーニングを要求した場合にのみ Cookie を削除し、Cookie の内容を読み取ったり記録したりしません。

### Q3: 設定データはサーバーにアップロードされますか？

**A:** いいえ。すべての設定データはローカルブラウザにのみ保存され、アップロードされることはありません。

### Q4: この拡張機能はインターネット接続が必要ですか？

**A:** いいえ。この拡張機能は完全にオフラインで動作し、ネットワーク接続は必要ありません。

### Q5: この拡張機能のプライバシーのお約束を確認するにはどうすればよいですか？

**A:** 以下の方法で確認できます：

1. オープンソースコードを表示：[https://github.com/soyof/clearCache](https://github.com/soyof/clearCache)
2. ブラウザの開発者ツールを使用してネットワークリクエストを監視（外部リクエストがないことがわかります）
3. ブラウザの権限使用記録を確認

### Q6: 拡張機能をアンインストールした後、データは残りますか？

**A:** いいえ。拡張機能をアンインストールすると、ローカルに保存されたすべての設定データが自動的に削除されます。

### Q7: この拡張機能は GDPR に準拠していますか？

**A:** はい。この拡張機能は個人データを収集しないため、GDPR およびその他のプライバシー規制に完全に準拠しています。
