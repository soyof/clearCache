## 3. Data Usage

### 3.1 Data Processing Methods

All functions of this Extension run on your local device:

1. **Cache Cleaning**: Directly calls browser API to delete cache, does not read cache content
2. **Cookie Cleaning**: Directly deletes cookies, does not read or record cookie values
3. **Storage Cleaning**: Clears LocalStorage and SessionStorage, does not access storage content
4. **History Cleaning**: Deletes browsing history, does not read or record history data
5. **Page Reload**: Triggers page refresh, does not access page content

### 3.2 Operations We Do NOT Perform

This Extension will **NEVER**:

- ‚ùå Send your data to any external servers
- ‚ùå Share your information with third parties
