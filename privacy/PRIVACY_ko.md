**최종 업데이트: 2025년 10월 21일**

## 소개

"캐시 클리너 어시스턴트"(이하 "본 확장 프로그램")를 사용해 주셔서 감사합니다. 저희는 귀하의 개인정보와 데이터 보안을 매우 중요하게 생각합니다. 본 개인정보 보호정책은 본 확장 프로그램이 정보를 수집, 사용, 저장 및 보호하는 방법을 이해하는 데 도움을 드리기 위한 것입니다.

**중요한 약속: 본 확장 프로그램은 개인 식별 정보를 외부 서버에 수집, 저장 또는 전송하지 않습니다. 모든 데이터 처리는 귀하의 로컬 장치에서 수행됩니다.**

## 1. 정보 수집

### 1.1 수집하지 않는 정보

본 확장 프로그램은 다음 정보를 **절대 수집하지 않습니다**:

- ❌ 개인 식별 정보(이름, 이메일, 전화번호 등)
- ❌ 브라우징 기록
- ❌ 웹사이트 방문 데이터
- ❌ 쿠키 내용
- ❌ 양식 데이터
- ❌ 비밀번호 또는 로그인 자격 증명
- ❌ 위치 정보
- ❌ 장치 식별자
- ❌ IP 주소

### 1.2 로컬에 저장되는 데이터

본 확장 프로그램은 개인화된 경험을 제공하기 위해 다음 설정 정보만 귀하의 로컬 브라우저에 저장합니다:

| 데이터 유형    | 목적                        | 저장 위치            |
| -------------- | --------------------------- | -------------------- |
| 언어 설정      | 선택한 인터페이스 언어 기억 | 로컬 브라우저 저장소 |
| 테마 설정      | 다크/라이트 테마 선택 저장  | 로컬 브라우저 저장소 |
| 알림 설정      | 알림 켜기/끄기 상태 기억    | 로컬 브라우저 저장소 |
| 정리 옵션      | 정리 기본 설정 저장         | 로컬 브라우저 저장소 |
| 예약 정리 계획 | 자동 정리 구성 저장         | 로컬 브라우저 저장소 |

**이러한 데이터는 귀하의 장치에만 저장되며 서버에 업로드되지 않습니다.**

## 2. 권한 설명

본 확장 프로그램은 핵심 기능을 제공하기 위해 다음 브라우저 권한이 필요합니다. 이러한 권한은 명시된 목적으로만 사용할 것을 약속합니다:

### 2.1 필수 권한

| 권한               | 목적                                            | 데이터 처리 방법                            |
| ------------------ | ----------------------------------------------- | ------------------------------------------- |
| `storage`          | 사용자 설정 및 기본 설정 저장                   | 로컬 저장소만, 업로드 안 함                 |
| `browsingData`     | 캐시, 쿠키 등 브라우징 데이터 지우기            | 삭제만 수행, 내용 읽지 않음                 |
| `tabs`             | 정리 작업을 수행하기 위해 현재 탭 정보 가져오기 | URL 도메인만 읽기, 전체 URL 기록 안 함      |
| `activeTab`        | 현재 탭에서 새로고침 작업 실행                  | 새로고침만 트리거, 페이지 내용 액세스 안 함 |
| `downloads`        | 다운로드 기록 지우기                            | 삭제만 수행, 파일 내용 액세스 안 함         |
| `history`          | 브라우징 기록 지우기                            | 삭제만 수행, 기록 내용 읽지 않음            |
| `cookies`          | 웹사이트 쿠키 지우기                            | 삭제만 수행, 쿠키 값 읽지 않음              |
| `unlimitedStorage` | 사용자 설정 저장(크기 제한 없음)                | 설정 데이터만 저장, 사용자 정보 수집 안 함  |
| `contextMenus`     | 컨텍스트 메뉴에 빠른 작업 추가                  | 메뉴 옵션만 제공, 데이터 수집 안 함         |
| `notifications`    | 작업 완료 알림 표시                             | 로컬에서만 표시, 외부로 전송 안 함          |
| `scripting`        | 페이지에서 정리 스크립트 실행                   | LocalStorage/SessionStorage만 지우기        |
| `<all_urls>`       | 모든 웹사이트에서 정리 작업 허용                | 정리에만 사용, 페이지 내용 액세스 안 함     |

### 2.2 권한 사용 원칙

- ✅ **최소 권한 원칙**: 기능 구현에 필요한 권한만 요청
- ✅ **투명한 사용**: 모든 권한의 목적은 본 문서에서 명확히 설명
- ✅ **로컬 처리**: 모든 데이터 처리는 로컬에서 완료
- ✅ **권한 남용 없음**: 명시된 목적 외에는 권한을 사용하지 않음

## 3. 데이터 사용

### 3.1 데이터 처리 방법

본 확장 프로그램의 모든 기능은 귀하의 로컬 장치에서 실행됩니다:

1. **캐시 정리**: 브라우저 API를 직접 호출하여 캐시 삭제, 캐시 내용 읽지 않음
2. **쿠키 정리**: 쿠키를 직접 삭제, 쿠키 값을 읽거나 기록하지 않음
3. **저장소 정리**: LocalStorage 및 SessionStorage 지우기, 저장소 내용 액세스 안 함
4. **기록 정리**: 브라우징 기록 삭제, 기록 데이터를 읽거나 기록하지 않음
5. **페이지 새로고침**: 페이지 새로고침 트리거, 페이지 내용 액세스 안 함

### 3.2 수행하지 않는 작업

본 확장 프로그램은 **절대 다음을 수행하지 않습니다**:

- ❌ 귀하의 데이터를 외부 서버로 전송
- ❌ 귀하의 정보를 제3자와 공유
- ❌ 귀하의 브라우징 행동 추적
- ❌ 귀하의 사용 습관 분석
- ❌ 광고 표시 또는 마케팅 수행
- ❌ 귀하의 데이터 판매 또는 임대

## 4. 데이터 보안

### 4.1 보안 조치

귀하의 데이터 보안을 보호하기 위해 다음 조치를 취합니다:

- 🔒 **로컬 저장소**: 모든 데이터는 귀하의 장치에만 저장
- 🔒 **네트워크 전송 없음**: 확장 프로그램에 네트워크 요청 코드 포함 안 됨
- 🔒 **외부 종속성 없음**: 제3자 서비스 또는 CDN에 의존하지 않음
- 🔒 **오픈 소스 투명성**: 소스 코드가 공개되어 검토 가능
- 🔒 **권한 격리**: 브라우저 보안 샌드박스 메커니즘 준수

### 4.2 데이터 삭제

다음 방법으로 본 확장 프로그램이 저장한 모든 데이터를 언제든지 삭제할 수 있습니다:

1. **확장 프로그램 제거**: 제거 후 모든 로컬 설정이 자동으로 삭제됩니다
2. **설정 재설정**: 확장 프로그램 설정에서 "기본 설정으로 복원" 클릭
3. **브라우저 데이터 지우기**: 브라우저 설정에서 확장 프로그램 데이터 지우기

### Q1: 이 확장 프로그램이 브라우징 기록을 수집합니까?

**A:** 아니요. 이 확장 프로그램은 귀하가 명시적으로 요청할 때만 브라우징 기록을 삭제하며 기록 내용을 읽거나 수집하지 않습니다.

### Q2: 이 확장 프로그램이 쿠키를 읽습니까?

**A:** 아니요. 이 확장 프로그램은 귀하가 정리를 요청할 때만 쿠키를 삭제하며 쿠키 내용을 읽거나 기록하지 않습니다.

### Q3: 설정 데이터가 서버에 업로드됩니까?

**A:** 아니요. 모든 설정 데이터는 로컬 브라우저에만 저장되며 업로드되지 않습니다.

### Q4: 이 확장 프로그램에 인터넷 연결이 필요합니까?

**A:** 아니요. 이 확장 프로그램은 완전히 오프라인으로 작동하며 네트워크 연결이 필요하지 않습니다.

### Q5: 이 확장 프로그램의 개인정보 보호 약속을 어떻게 확인할 수 있습니까?

**A:** 다음 방법으로 확인할 수 있습니다:

1. 오픈 소스 코드 보기: [https://github.com/soyof/clearCache](https://github.com/soyof/clearCache)
2. 브라우저 개발자 도구를 사용하여 네트워크 요청 모니터링(외부 요청이 없음을 확인할 수 있습니다)
3. 브라우저의 권한 사용 기록 확인

### Q6: 확장 프로그램을 제거한 후 데이터가 남아 있습니까?

**A:** 아니요. 확장 프로그램을 제거하면 로컬에 저장된 모든 설정 데이터가 자동으로 삭제됩니다.
