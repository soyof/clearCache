# 隐私政策 (Privacy Policy)

**最后更新日期：2025 年 10 月 21 日**

---

## 简介

感谢您使用"缓存清理助手"（以下简称"本扩展"）。我们非常重视您的隐私和数据安全。本隐私政策旨在帮助您了解本扩展如何收集、使用、存储和保护您的信息。

**重要承诺：本扩展不会收集、存储或传输任何个人身份信息到外部服务器。所有数据处理均在您的本地设备上完成。**

---

## 1. 信息收集

### 1.1 我们不收集的信息

本扩展**不会**收集以下任何信息：

- ❌ 个人身份信息（姓名、邮箱、电话等）
- ❌ 浏览历史记录
- ❌ 网站访问数据
- ❌ Cookie 内容
- ❌ 表单数据
- ❌ 密码或登录凭证
- ❌ 地理位置信息
- ❌ 设备标识符
- ❌ IP 地址

### 1.2 本地存储的数据

本扩展仅在您的本地浏览器中存储以下设置信息，用于提供个性化体验：

| 数据类型     | 用途                  | 存储位置       |
| ------------ | --------------------- | -------------- |
| 语言偏好     | 记住您选择的界面语言  | 本地浏览器存储 |
| 主题设置     | 保存深色/浅色主题选择 | 本地浏览器存储 |
| 通知设置     | 记住通知开关状态      | 本地浏览器存储 |
| 清理选项     | 保存您的清理偏好设置  | 本地浏览器存储 |
| 定期清理计划 | 存储自动清理配置      | 本地浏览器存储 |

**这些数据仅存储在您的设备上，永远不会被上传到任何服务器。**

## 2. 权限说明

本扩展需要以下浏览器权限来提供核心功能。我们承诺仅将这些权限用于其声明的用途：

### 2.1 必需权限

| 权限               | 用途                             | 数据处理方式                       |
| ------------------ | -------------------------------- | ---------------------------------- |
| `storage`          | 保存用户的设置和偏好             | 仅本地存储，不上传                 |
| `browsingData`     | 清理缓存、Cookies 等浏览数据     | 仅执行删除操作，不读取内容         |
| `tabs`             | 获取当前标签页信息以执行清理操作 | 仅读取 URL 域名，不记录完整 URL    |
| `activeTab`        | 在当前标签页执行重载操作         | 仅触发刷新，不访问页面内容         |
| `downloads`        | 清理下载历史记录                 | 仅执行删除操作，不访问文件内容     |
| `history`          | 清理浏览历史记录                 | 仅执行删除操作，不读取历史内容     |
| `cookies`          | 清理网站 Cookies                 | 仅执行删除操作，不读取 Cookie 值   |
| `unlimitedStorage` | 存储用户设置（无大小限制）       | 仅存储设置数据，不收集用户信息     |
| `contextMenus`     | 在右键菜单中添加快捷操作         | 仅提供菜单选项，不收集数据         |
| `notifications`    | 显示操作完成通知                 | 仅本地显示，不发送到外部           |
| `scripting`        | 在页面中执行清理脚本             | 仅清理 LocalStorage/SessionStorage |
| `<all_urls>`       | 允许在所有网站上执行清理操作     | 仅用于清理，不访问页面内容         |

### 2.2 权限使用原则

- ✅ **最小权限原则**：仅请求实现功能所必需的权限
- ✅ **透明使用**：所有权限用途在此文档中明确说明
- ✅ **本地处理**：所有数据处理均在本地完成
- ✅ **不滥用权限**：不将权限用于声明之外的任何目的

## 3. 数据使用

### 3.1 数据处理方式

本扩展的所有功能均在您的本地设备上运行：

1. **缓存清理**：直接调用浏览器 API 删除缓存，不读取缓存内容
2. **Cookie 清理**：直接删除 Cookie，不读取或记录 Cookie 值
3. **存储清理**：清除 LocalStorage 和 SessionStorage，不访问存储内容
4. **历史清理**：删除浏览历史，不读取或记录历史数据
5. **页面重载**：触发页面刷新，不访问页面内容

### 3.2 不进行的操作

本扩展**绝不会**：

- ❌ 将您的数据发送到任何外部服务器
- ❌ 与第三方共享您的信息
- ❌ 跟踪您的浏览行为
- ❌ 分析您的使用习惯
- ❌ 投放广告或进行营销
- ❌ 出售或出租您的数据

## 14. 常见问题 (FAQ)

### Q1: 本扩展会收集我的浏览历史吗？

**A:** 不会。本扩展只会在您明确要求时删除浏览历史，不会读取或收集历史记录内容。

### Q2: 本扩展会读取我的 Cookie 吗？

**A:** 不会。本扩展只会在您要求清理时删除 Cookie，不会读取或记录 Cookie 的内容。

### Q3: 我的设置数据会被上传到服务器吗？

**A:** 不会。所有设置数据仅存储在您的本地浏览器中，永远不会被上传。

### Q4: 本扩展需要联网吗？

**A:** 不需要。本扩展完全离线工作，不需要任何网络连接。

### Q5: 如何验证本扩展的隐私承诺？

**A:** 您可以：

1. 查看开源代码：[https://github.com/soyof/clearCache](https://github.com/soyof/clearCache)
2. 使用浏览器开发者工具监控网络请求（您会发现没有任何外部请求）
3. 查看浏览器的权限使用记录

### Q6: 卸载扩展后数据会保留吗？

**A:** 不会。卸载扩展后，所有本地存储的设置数据都会被自动删除。

### Q7: 本扩展符合 GDPR 吗？

**A:** 是的。由于本扩展不收集任何个人数据，因此完全符合 GDPR 和其他隐私法规的要求。
